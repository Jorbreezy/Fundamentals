# Fundamentals of webdev
A project focused on learning the fundamentals of webdev.

1. Javascript Fundamentals
2. REST API Fundamentals
3. GraphQL Fundamentals
4. Typescript Fundamentals

# Documentation

## Getting Started

### Install Dependencies
```
npm install
``` 
### Run dev server
```
npm run dev
``` 
### build project
```
npm run build
```
### Run build server
```
npm start
```
